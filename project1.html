Maret 2024 Web Application

Web, Domain & Hosting Mandiri

Proyek pembuatan website koperasi sekolah dengan server, hosting, dan domain yang dikelola sendiri. Website ini menyediakan sistem informasi serta layanan pemesanan merchandise, print, dan fotokopi secara online untuk siswa dan guru.

Ringkasan Proyek

Proyek ini saya buat sebagai simulasi sekaligus implementasi nyata dalam membangun hosting mandiri. Mulai dari setup server, konfigurasi domain, hingga deployment website dilakukan secara independen.

Website difokuskan untuk mendukung aktivitas koperasi sekolah, khususnya pemesanan merchandise, layanan print, dan fotokopi online agar lebih praktis bagi siswa maupun guru.

  • Hosting Mandiri: Setup server web secara langsung
  • Domain Sendiri: Konfigurasi DNS dan akses publik
  • Sistem Informasi: Informasi produk koperasi sekolah
  • Pemesanan Online: Print, fotokopi, dan merchandise
  • Database Management: Penyimpanan data produk & pesanan

Jurnal Pengembangan

Tahap 1 - Setup Server

Instalasi dan konfigurasi web server serta database. Fokus pada stabilitas server dan keamanan akses.

Tahap 2 - Domain & Hosting

Konfigurasi domain sendiri, DNS, dan integrasi hosting agar website bisa diakses secara online.

Tahap 3 - Pengembangan Website

Pembuatan tampilan website koperasi sekolah serta sistem pemesanan sederhana untuk merchandise dan layanan cetak.

Tahap 4 - Testing

Pengujian akses website, database, dan performa server agar sistem berjalan stabil.

Hari 8 - Testing & Polish

Testing berbagai edge cases, fixing bugs, dan improving code quality. Menambahkan error handling dan validasi input. Optimize performance dan menambahkan responsive design untuk mobile devices.
